WooCommerce integration for VHL Europe shipping services (DPD, Packeta)

== Description ==

VHL Europe WooCommerce plugin integrates your WooCommerce store with VHL Europe fulfillment services. The plugin supports DPD and Packeta shipping methods with automatic order processing.

= Features =

* DPD shipping integration
* Packeta shipping integration (requires official Packeta plugin)
* Cash on Delivery (COD) support
* Automatic order submission to VHL Europe API
* Mapping of WooCommerce shipping/payment methods
* Admin configuration panel

= Requirements =

* WooCommerce 5.0+
* PHP 7.4+
* VHL Europe API credentials

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/vhl-europe-woocommerce/`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to WooCommerce > VHL Europe to configure the plugin
4. Enter your API Bearer Token
5. Configure shipping and payment method mappings

== Frequently Asked Questions ==

= Do I need the official Packeta plugin? =

Yes, if you want to use Packeta shipping, you need to install the official Packeta plugin from WordPress.org.

= Where do I get the API Bearer Token? =

Contact VHL Europe to obtain your API credentials.
